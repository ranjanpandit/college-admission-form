# Colorful Registration (Next.js App Router + Tailwind)

This is a ready-made demo project containing a JSON-driven **multi-tab registration** page.

## Features
- Next.js (App Router)
- TailwindCSS (configured)
- JSON file (`/data/registration.json`) controls tabs & fields
- Glassmorphism UI, gradient background
- Icons via `@iconify/react`
- File inputs, select, textarea, date, number, email fields supported

## Setup (Windows / Linux / macOS)

1. Extract the ZIP.
2. Open a terminal in the project folder.
3. Run:
```bash
npm install
npm run dev
```

4. Open `http://localhost:3000/register`

## Notes
- This package does not include `node_modules`. `npm install` will fetch dependencies.
- If Tailwind CLI issues persist on your machine, installing dependencies locally and running `npm run dev` is the recommended approach.
